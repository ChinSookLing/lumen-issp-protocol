# Q1 Tabletop Drill Facilitator Script

**Owner:** Node-02-Bing
**Date:** 2026-03-15 (待 Tuzi 確認最終日期)
**來源:** M86 Node-02-Bing 回覆

## Flow

- **Opening (10m)**: Welcome, objectives, confirm roles.
- **Scenario Brief (20m)**: Present canary report + timeline.
- **Decision Roleplay (30m)**: SRE validate → Owner recommend → Legal assess → Chair call vote.
- **Technical Validation (20m)**: Tech Lead simulate fallback, Recorder log.
- **Communications (20m)**: Messaging owner draft notify + external statement.
- **AAR (20m)**: Recorder capture what worked/failed, assign action items.

## Facilitator Prompts

- "Validate metrics now."
- "Owner, state recommendation."
- "Legal, provide input."
- "Chair, call decision."
- "Tech Lead, execute fallback."
- "Recorder, confirm record fields."
- "Observers, note deviations."

## Timing Cues

- Announce 5m remaining per phase.
- Keep decision window ≤5m.
- Mark simulated steps clearly in record.
