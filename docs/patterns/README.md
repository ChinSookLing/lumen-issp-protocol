# Patterns

This folder contains L1/L2 design documents (pattern specs, mutex rules, activation gates, PDDs).

Patterns are NOT the same as the Six-Dimension defense map.
- Governance/strategy artifacts live under: `docs/dimensions/`
- Cross-dimension governance policies live under: `docs/governance/`
- Charter/redlines live under: `docs/charter/`

If you are looking for risk ownership and gaps, start from `docs/dimensions/README.md`.
