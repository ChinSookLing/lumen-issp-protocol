# 對外 FAQ 架構 v0.1

Status: Draft

Author: Node-03

---

1. **Lumen 是什麼？** 開源協議，偵測結構性操控，不存原始對話，不輸出身份指控。
2. **會取代人類判斷嗎？** 不會。只輸出結構描述，不給行動建議。
3. **會存資料嗎？** 預設只存 fingerprint，90 天自動刪除。
4. **是開源的嗎？** 是，Apache 2.0。
5. **會判定誰對誰錯嗎？** 不。像氣象預報說有風暴，不說誰壞。
6. **可以監控員工嗎？** Charter 禁止用於懲戒、羞辱、排名。
7. **跟其他工具有何不同？** 結構化規則，非 ML 分類，可解釋性強。
8. **如何防濫用？** 紅線 + 治理流程 + 開源可審計。
9. **支援哪些語言？** 中英完整，馬來語/日語擴充中。核心 pattern 語言無關。
10. **如何貢獻？** 見 CONTRIBUTING_mapping.md。
