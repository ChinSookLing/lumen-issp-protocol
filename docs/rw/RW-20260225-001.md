# RW-20260225-001 — OpenClaw × Google OAuth 高頻代理 → 帳號封鎖

* Date: 2026-02-25
* Category: Adapter Engineering / Rate Limit / OAuth Risk
* Group: A (Adapter 工程)
* Layer: L4

## Summary
第三方工具透過 OAuth 高頻呼叫 → 觸發平台風控 → 帳號永久停權。
Adapter 可能無意間形成「高頻代理」，造成使用者帳號被封。

## Failure Modes
- FM-01: 無限流（no rate limit）
- FM-02: 無預算（no budget cap）
- FM-03: 無熔斷（no circuit breaker）
- FM-04: 無退避（no backoff/jitter）

## Lumen Improvement
- Adapter Safety Envelope: rate limit / budget / backoff / circuit breaker / fail-closed / HITL
- UI Contract 加 `connector_budget` 欄位
- L4 Export 增補 `connector.calls_count` / `connector.circuit_state`

## DoD
- T-01: rate limiting enforcement
- T-02: backoff + jitter
- T-03: circuit breaker trigger
- T-04: fail-closed default
- T-05: token hygiene (no raw token in log)
- T-06: budget enforcement

## Status
❌ Adapter 無限流、無預算、無熔斷、無退避 — 需 L4 hardening
