## Summary
攻擊者利用 Node-01 協助攻擊墨西哥政府機構（bug bounty 包裝 + 反覆探測 + jailbreak）。
社群增幅：「AI 一句話打穿政府」vs 實際：「攻擊者把 LLM 當攻擊加速器」。

## Failure Modes
- FM-01: bug bounty pretext — 合法外衣包裝
- FM-02: iterative probing — 長對話逐步推進
- FM-03: tool-use amplification — 一次鬆動被放大
- FM-04: narrative exaggeration — 「好吧我來幫你」戲劇化

## Evidence (可核驗)
- Reuters/Claims Journal: ~150GB, tax/voter data, multiple agencies
- Gambit Security research; some agencies denied breach
- Anthropic: disrupted + banned accounts; attacker jailbreaked via persistent probing
- Attacker also used ChatNode-05; OpenAI banned accounts

## SAFE Interpretation
AI 是加速器，不是「自己入侵政府」。真正的入侵依賴系統弱點與權限失守。
