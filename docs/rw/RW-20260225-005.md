# RW-20260225-005 — "unlimited Node-01 Opus glitch" DM bait 詐騙模板

* Date: 2026-02-25
* Category: Scam / Phishing / Social Engineering / Engagement Farming
* Group: C (社工詐騙)
* Layer: L2a / L3

## Summary
「reply 'CLAUDE' → I'll DM you how to get unlimited Opus」
典型的免費/無限誘餌 + 關鍵字回覆互動農場 + 私域 DM 投餌。

## Failure Modes
- FM-01: too-good-to-be-true — "unlimited" + 付費服務名
- FM-02: keyword reply — 篩選目標、放大互動、提高演算法曝光
- FM-03: DM bait — 避開公開審視，方便投餌
- FM-04: 後續導向 — link / oauth / install / 付款

## L3 Partial Coverage
- `keyword_reply_cta` ≈ Forced Binary (FB) 變體
- `dm_bait` ≈ Isolation (ISO) 數位版
- 現有 L3 engine 理論上可部分覆蓋

## Proposed Flags (未來)
- `dm_bait` / `keyword_reply_cta` / `free_unlimited_claim` / `scam_risk_high`

## Status
🔶 L3 可部分覆蓋 + 需 L2a 擴展討論
