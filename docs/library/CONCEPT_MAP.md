# Concept Map — Patterns ↔ Components ↔ Reference Concepts (v0.1)

> 用途：統一語言。
> 注意：此表是「概念對照」，不是模型證據；證據以 TR/TRS/CI 為準。

| Pattern | Components (internal) | Concept anchors (references) | Notes (boundary) |
|--------|------------------------|------------------------------|------------------|
| EP | fear narrative / forced response / shame escalation | Fear appeal & risk/outage framing (WITTE-1992, SANDMAN-1987) | 區分「合理警示」vs「情緒施壓」 |
| FC | binary_frame / consequence / closure_pressure / urgency_signal | Scarcity & forced choice pressure (CIALDINI-2009, KAHNEMAN-2011) | 合法 deadline ≠ 虛假二選一 |
| MB | loyalty test / moral membership / "prove you care" | Commitment/consistency & social proof dynamics (CIALDINI-2009) | "價值觀表態"不必然是操控 |
| GC | authority claim / dismissal / infallibility framing | Authority principle + status coercion (CIALDINI-2009) | 專業建議 ≠ 不可質疑 |
| DM | debt framing / withdrawal threat / guilt leverage | Reciprocity pressure & coercive control (CIALDINI-2009, BANCROFT-2002, STARK-2007) | 自我內疚（self-guilt）≠ 對人施壓 |
| EA | bond_claim / isolation_hint / emotional_debt / attachment_pressure | Attachment vs coercive bonding (BOWLBY-1969, STARK-2007, BANCROFT-2002) | "我想你"不等於情感錨定操控 |
| IP | identity probing / boundary crossing | Boundary & privacy norms (NIST-AI-RMF-2023) | 身份詢問需看情境與權力差 |
| VS | virtue signaling / moral posturing | Social identity & signaling (CIALDINI-2009) | 正常道德表達 ≠ 操控 |
| Class-0 | vacuum / structural abandonment | Risk governance baseline (NIST-AI-RMF-2023) | 目標是避免誤咬與過度推斷 |
| SAFETY_CRITICAL (node policy) | HITL / delay-first / audit | Safety lifecycle mindset (IEC-61508) | 不進控制回路、先保守 |
| Provenance (meta) | build_fingerprint / reproducibility | SLSA / reproducible builds (SLSA-2023, REPRODUCIBLE-BUILDS) | 可追溯輸出是治理核心 |
