## Official Logos

### Primary: L4 — Minimal Blue Shield

- **Usage:** All official documents, repo headers, GitHub, website, favicons, social media avatars, printed materials
- **Description:** Deep blue shield with white double-line border, four-point star at center, LUMEN ISSP in white
- **Key strength:** Maximum legibility at any size (4K banner to 32px favicon)

### Secondary: L6 — Hexagonal Core

- **Usage:** Videos, animations, promotional materials, dynamic banners
- **Description:** 3D hexagonal frame with circuit-board pattern, glowing cyan core radiating outward
- **Key strength:** Strong tech aesthetic, animation potential

## Clear Space and Minimum Size

- Clear space: 1/2 of shield width
- Web: min 48px height
- Mobile/Favicon: min 32px
- Print: min 15mm
- Business card: min 20mm

---

## Usage Rules

### Permitted
- Maintain original proportions and colors
- Use on white or dark blue backgrounds
- Display with subtitle below
- Favicon uses simplified version

### Prohibited
- Changing shield color
- Stretching, tilting, adding shadows
- Complex backgrounds without solid backing
- Modifying star shape
- Separating LUMEN and ISSP
- Using L6 where L4 should be used
